# UI推理工具 - 快速开始指南

## 5分钟快速上手

### 第一步：准备文件

确保以下文件存在：
- ✅ `dist/ui_inference_dist/ui_inference.exe` - 可执行文件
- ✅ `checkpoints/seg/ui/sfnet_res101_ui_latest.pth` - 模型权重文件
- ✅ `testimage/*.png` - 测试图片（任意PNG图片）

### 第二步：运行测试

在项目根目录打开命令行，运行：

```batch
.\scripts\test_exe.bat
```

### 第三步：查看结果

打开输出目录查看结果：
```
dist\ui_inference_dist\test_output\output.html
```

## 常用命令

### 处理单张图片

```batch
cd dist\ui_inference_dist
ui_inference.exe --image "图片路径.png" --config "configs\seg\sfnet_res101_ui.conf" --checkpoint "checkpoints\seg\ui\sfnet_res101_ui_latest.pth" --output "输出目录" --gpu -1
```

### 参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| `--image` | 输入图片路径 | `--image "test.png"` |
| `--config` | 配置文件路径 | `--config "configs\seg\sfnet_res101_ui.conf"` |
| `--checkpoint` | 模型文件路径 | `--checkpoint "checkpoint.pth"` |
| `--output` | 输出目录 | `--output "output"` |
| `--gpu` | GPU ID（-1=CPU） | `--gpu -1` |

---

## 输出文件

运行成功后，在输出目录中会生成：
- `output.html` - HTML代码文件
- `prediction_mask.png` - 分割结果可视化
- 原始图片副本

## 常见问题

**Q: 程序无法运行？**
A: 确保使用命令行运行，查看错误信息。

**Q: 找不到配置文件？**
A: 使用相对路径 `configs\seg\sfnet_res101_ui.conf`。

**Q: 运行很慢？**
A: 使用CPU模式（`--gpu -1`）速度较慢，这是正常的。

**Q: 支持中文路径吗？**
A: 支持！程序会自动处理中文路径。
